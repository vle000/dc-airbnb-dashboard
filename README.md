# Washington DC AirBnb Dashboard

link: https://dc-airbnb-dashboard-aml6xrxnfrj8p4mwbqbxnf.streamlit.app/

This dashboard analyzes data regarding Airbnb listings within the District of Columbia and visualizes them into insightful maps, charts, and diagrams.
It uses data from thise Kaggle database: https://www.kaggle.com/datasets/thedevastator/airbnb-listings-and-reviews-in-washington-dc

The initial top insights give the total listings, # of unique neigborhoods, average price, and the raw table of the dataset.
The following visualizations are listed below:
  *Airbnb Listing Density Map
  *Top 10 Neighborhoods with the most listings (bar chart)
  *Average Price By Neighborhood (bar chart)
  *Price Distribution per Room Type (dot chart)
  *Price and minimum nights comparisons (dot chart)
  *Listings and price distributions 1km from various DC attractions (table and boxplot)

NOTE: This dashboard is not affliated with Airbnb or the city of Washington
